# Cobblemon Mouse Menu

Press Left Alt (Bindable to another key) to enable mouse cursor mode (camera locked) to start using the mod.

Features list:
1. Cobblemon party list at the left side of the screen
 - Left click on a Cobblemon in the party list to **QUICK INSPECT**.
 - Right click on a Cobblemon in the party list to display Context Menu, quickly: 
   Summon/Recall, Take/Give item.
2. Left click on an **RCT Trainer NPC**.
 - **Display that Trainer's pokemon list and complete stats**.
 - The trainer inspector has an "All" toggle → a **full-team sheet**
   (every Pokémon's types, ability, nature, item, moves, and final stats) 
   designed to screenshot and share quickly.
   (Without RCT, this feature will not function, but other features will still function.)
3. Left click on YOUR owned already-summoned Cobblemon to **QUICK INSPECT**.

Requires: Cobblemon · Fabric API (+ RCT optional, for overworld trainers)
