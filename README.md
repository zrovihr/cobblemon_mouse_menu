# Cobblemon Mouse Menu

Hold Left Alt (Bindable to another key) to enable mouse cursor mode (camera locked) to start using the mod.

Features list:
1. Cobblemon party list at the left side of the screen
 - Left click on a Cobblemon in the party list to *QUICK INSPECT*.
 - Right click on a Cobblemon in the party list to display Context Menu, quickly: Summon/Recall, Take/Give item, 
2. Left click on an RCT Trainer NPC.
 - *Display that Trainer's pokemon list and complete stats*.
   Required to work: https://modrinth.com/mod/rctmod , AND this mod (cobblemon_mouse_menu) to be installed both from SERVER and CLIENT.
   Otherwise this feature will not function. This is because the Trainer's Cobblemon data is not exposed to the Client,
   so we need the Server's permission to do so.
3. Left click on an already summoned Cobblemon to *QUICK INSPECT*.
