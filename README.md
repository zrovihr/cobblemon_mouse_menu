# What is this mod?

Allows you to use Keys on your keyboard to select
Cobblemon battle HUD buttons (Fight, Switch, ),
as well as Selecting the skills and select a Pokemon to switch

# Keys
'`' (Tilde): BACK
'1', '2', '3', '4': Select TopLeft, TopRight, BottomLeft, BottomRight.
'1', '2', '3', '4', '5', '6': Switching Pokemon, Slots 1 ~ 6

They keys will only active when the Battle HUD is active.

# Installation
Download the .zip file from Release, extract the .jar file in /mods folder. Just like normal mods.
